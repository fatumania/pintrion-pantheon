Подключение к WhatsApp

Только WhatsApp требует привязки аккаунта. Telegram, Viber, Signal и IMO работают без подключения — просто вводите номер и проверяйте.

Нажмите "Подключить" и отсканируйте QR код в WhatsApp (Настройки - Привязанные устройства)

WhatsApp — нажмите "Подключить", отсканируйте QR код телефоном
Telegram — работает сразу, проверка через API
Viber — работает сразу, проверка через API
